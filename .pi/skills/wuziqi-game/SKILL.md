# Arena 五子棋规则

1. 现在进行的是一个五子棋游戏。执子不固定；你在使用该 skill 时会被明确指定为 `X` 或 `O`，另一方视为对手。
2. 只根据 `Arena/五子棋.md` 文件发生的修改来进行：如果该文件发生修改，读取棋盘并判断是否需要落子；否则其他文件或文件夹的修改一律忽略。
3. 先判断是否应行动：如果当前不是你行动（例如刚好是你上一次落子产生的回流修改），请回复 `<|no_change|>`。
4. 如果棋局已结束，不要继续落子；请直接说明游戏结果（谁胜或平局），并引导开始下一局。
5. 如果棋盘非法，不要落子；请直接报告非法原因（如棋子数量不匹配、多人同时成五、表格格式损坏等），并引导先修正棋盘。
6. 需要行动时，只在一个空位落一个“你当前被指定的棋子（`X` 或 `O`）”，并保持 Markdown 表格结构和其他内容不变（只修改一个格子）。
7. 合法性和胜负判断按 15x15 五子棋标准执行：任一方在横、竖、主对角线、副对角线方向形成连续 5 子（或以上）即胜。
8. 落子优先级：先立即获胜（己方） > 阻止对手下一手获胜 > 优先中心区域（H8 附近） > 优先与己方已有棋子相邻的位置 > 其余空位。
