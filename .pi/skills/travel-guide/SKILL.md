# 旅游攻略网页生成

## 工作流程

### 1. 明确需求

向用户确认：
- 目的地（城市/景区）
- 出发地（影响交通方案）
- 出行时间和天数
- 预算范围（穷游/中等/高端）
- 人数和同行人（亲子/情侣/朋友）
- 特殊偏好（美食优先、打卡拍照、自然风光等）

### 2. 信息收集

使用虚拟浏览器（gui）在网站内实际操作搜索与浏览信息（例如 `open_url_in_browser` 打开网站，`click` 聚焦搜索框，`type` 输入关键词，`press`/`hotkey` 提交搜索，`scroll` 浏览结果）。

**小红书（xiaohongshu.com）**— 真实体验和避坑建议：
- 在小红书站内搜索 `{目的地} 旅游攻略`
- 在小红书站内搜索 `{目的地} 避坑`
- 在小红书站内搜索 `{目的地} 美食推荐`
- 提取：热门景点排序、真实花费、拍照点、避坑提醒、相关景点和攻略图片

**携程（ctrip.com / trip.com）**— 价格和预订信息：
- 在携程站内搜索 `{目的地} 酒店` 获取住宿价格区间
- 在携程站内搜索 `{出发地} 到 {目的地} 机票` / `{出发地} 到 {目的地} 火车票` 获取交通价格
- 提取：酒店价格区间、航班/高铁时刻和价格、门票价格、相关酒店房间图片

**马蜂窝（mafengwo.cn）**— 行程规划参考：
- 在马蜂窝站内搜索 `{目的地} 行程 {天数}天`
- 提取：经典行程路线、景点游览时长建议、相关景点和攻略图片

**大众点评（dianping.com）**— 美食排名：
- 在大众点评站内搜索 `{目的地} 必吃`
- 提取：高分餐厅、人均消费、招牌菜、相关美食图片

**补充搜索**：
- 在浏览器中打开搜索引擎，搜索 `{目的地} 天气 {月份}` — 穿衣建议
- 在浏览器中打开搜索引擎，搜索 `{目的地} 交通攻略` — 市内交通（地铁/公交/打车）
- 通过 `image_search` 搜索和旅行目的相关的图片补充在攻略中。

### 3. 生成网页

生成的网页可以考虑包含以下主题内容

| 区块 | 内容 |
|------|------|
| 概览 | 最佳旅行季节、建议天数、预算概览 |
| 交通 | 大交通（机票/高铁价格对比表）+ 市内交通方式 |
| 住宿 | 按预算分档推荐（含价格区间和地段建议） |
| 景点 | 必去景点列表（门票、建议游览时长、tips） |
| 美食 | 必吃餐厅/小吃（人均、推荐菜、地址） |
| 行程 | 按天规划的详细行程（每天的景点+餐饮+交通） |
| 贴士 | 避坑提醒、省钱技巧、注意事项 |
| 预算 | 总费用估算表（交通+住宿+餐饮+门票+其他） |
